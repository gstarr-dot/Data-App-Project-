Overview: \
This is a Streamlit-based personal finance tracker that allows users to log, filter, and visualize their spending habits. 

How to Run the App: \
-Install dependencies: pip install streamlit pandas matplotlib \
  pandas for organizing, filtering, and analyzing spending data. \
  matplotlib for the bar and pie charts

-In the terminal: py -m streamlit run "Spending App.py"

Features: 
- Add expenses with amount, category, and date
- Data is saved to the JSON file, and the app reads, displays, filters, and updates entries. 
- Automatically generates spending insights such as total spending, top spending category, and the most expensive purchase. 
- Filter spending by category and minimum amount
- View spending history in a table, bar chart, and pie chart. 
- Track budget 
- Delete individual entries


Future Improvements: 
- Add a login system
- Make categories more specific
- Monthly Spending Breakdowns
