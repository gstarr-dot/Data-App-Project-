Overview: \
This is a Streamlit-based personal finance tracker that allows users to log, filter, and visualize their spending habits. 

Features: 
- Add expenses with amount, category, and date
- Filter spending by category and minimum amount
- View spending history in a table
- Track budget usage with a progress bar
- Delete individual entries 

How to Run the App: \
-Install dependencies: pip install streamlit pandas metplotlib

-Run the app: py -m streamlit run "Spending App.py"
